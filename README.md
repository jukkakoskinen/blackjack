# Blackjack

A simple game of Blackjack made using Odin and Raylib.

![Blackjack](.github/screenshots/1.png)

## Running

odin run src -out:bin/blackjack

## Controls

| Key | Action |
|-----|--------|
| **H** | Hit |
| **S** | Stand |
| **R** | Restart |
| **Esc** | Quit |
